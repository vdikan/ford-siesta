write_rho read_rho check_rho

To have both reading and writing, allowing a change in the values of variables used as array dimensions, is not very safe. Thus, the old iorho has been split in three routines:

  1. write_rho: Writes grid magnitudes to file
  2. read_rho: Reads grid magnitudes from file
  3. check_rho: Checks the appropriate dimensions for reading.

The magnitudes are saved in SINGLE PRECISION (sp), regardless of the internal precision used in the program. Historically, the internal precision has been "single".
